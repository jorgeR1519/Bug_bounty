# Hallazgo Informativo (P5) del VDP del Tesoro

## Resumen

Este repositorio documenta un hallazgo de seguridad presentado a través del Programa de Divulgación de Vulnerabilidades (VDP) del Departamento del Tesoro de EE. UU. mediante Bugcrowd.

El hallazgo se refería a la ausencia de controles de limitación de velocidad observables en un flujo de trabajo de restablecimiento de contraseña.

## Resultado

- Programa: VDP del Departamento del Tesoro de EE. UU.
- Plataforma: Bugcrowd
- Clasificación: Informativo (P5)
- Estado: Revisado y reproducido

## Habilidades demostradas

- Pruebas de seguridad de aplicaciones web
- Burp Suite
- Análisis de solicitudes HTTP
- Divulgación responsable
- Informes de vulnerabilidades
